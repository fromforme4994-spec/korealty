# 코리얼티디엠씨㈜ 기업 사이트

정적 사이트입니다. 빌드 과정이 없으며, 파일을 그대로 올리면 동작합니다.

## 구성

```
index.html      홈 (히어로 영상 · 사업영역 인터랙션 · 현장 지도)
about.html      회사소개 (개요 · CEO 메시지 · 연혁 · 조직)
business.html   사업영역 (개발 · 마케팅 · 컨설팅 · 자산관리)
portfolio.html  실적 (서울 지도 + 28건 목록, 유형별 필터)
contact.html    문의 (전화 · 메일 · 팩스 · 오시는 길)
404.html        없는 주소

js/data.js      프로젝트 28건 + 사업영역 4축 (내용 수정은 여기서)
js/map.js       서울 25개 자치구 경계 + 현장 좌표
js/main.js      인터랙션
css/style.css   전체 스타일
```

## 내용을 고칠 때

- **실적·사업영역 문구**: `js/data.js`
- **현장 위치(지도 핀)**: `js/map.js`의 `SITE_XY` — 값은 1000x620 SVG 좌표계 기준
- **회사 정보(주소·전화)**: 각 HTML의 푸터, `index.html`의 JSON-LD

## 글자가 네모로 깨져 보인다면

폰트는 **사이트에 실제로 쓰인 글자만** 담아 8MB → 0.3MB로 줄여 두었습니다.
새로운 문구를 넣어 없던 글자가 생기면 다시 만들어야 합니다.

```
python3 tools/subset-fonts.py
```

원본 OTF는 `assets/fonts/src/`에 있으며 배포에는 포함되지 않습니다(.gitignore).
